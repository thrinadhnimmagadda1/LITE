LITE Backend API

Welcome to the LITE (Literature Insight and Text Exploration) Backend API.

Available Endpoints

GET /api/papers/

Retrieve a list of research papers with optional pagination.

Query Parameters:

  • page - Page number (default: 1)
  • page_size - Number of items per page (default: 20, max: 100)

Example: GET /api/papers/?page=1&page_size=10

POST /api/search-terms/

Update search terms and trigger paper processing.

Request Body (JSON):

{
    "query": "machine learning",
    "keywords": "deep learning, neural networks"
}

GET /admin/

Access the admin interface (admin credentials required).

For more information, please refer to the GitHub repository.
